# React Learning Repository

This repository documents my learning journey while studying *React.js*.  
It contains both *hands-on React implementations* and *detailed Markdown notes* written while learning the concepts.

The goal of this repository is to build a strong understanding of *modern frontend development using React*, covering topics such as component architecture, hooks, routing, state management, UI design, and testing.
